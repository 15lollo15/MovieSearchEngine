Per il corretto funzionamento del programma è necessario aver installato la libreria 'whoosh'

Per eseguire il programma, da riga di comando, digitare 'python server.py", se il programma parte con successo
dovrebbe aprirsi una finestra del browser dove sottoporre le query.
---------------------------------------------------------
Per eseguire il benchmark, una volta avviato il programma, digitare su un nuovo terminale "python benchmark.py". Partirà un
programma testuale con spiegazione in linguaggio umano della query e la query stessa. Per procedere alla query successiva
digitare 'c'. Il programma di volta in volta aprità una nuova scheda del browser con i risultati della query descritta.
